4. What’s the ​difference​ between ArrayList and LinkedList? Try to explain in Mandarin.
ArrayList是將資料從放在一個連續的記憶體，
LinkedList除了記錄本身儲存的資料，還儲存前一筆與後一筆物件的reference，可以分散放在不同區塊的記憶體。

如果需要頻繁取List內的資料，選擇ArrayList會比較好，他可以從他的(index)直接取資料，
LinkedList則必須從開始的結點一路找到所要值的位置所以耗時。

如果需要頻繁新增or刪除資料，則使用LinkedList較好，他指需要更改所以新增or刪除的資料其前後一筆資料所儲存的結點位置，
用ArrayList的話新增or刪除資料後，還必須處理搬動新增or刪除資料之後的所有其他資料的位置index，效率較差。
